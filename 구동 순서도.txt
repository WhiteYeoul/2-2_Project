status[0] = 각 테이블 서빙 or 회수 상태 표시
status[1] = 쟁반 테이블 들림 확인 (안들림 = 0, 들림 = 1)
status[2] = 쟁반 상태(식탁에 없음 = 0, 식탁에 있음 = 1)

[서빙]
1. 관리자가 이동 버튼 누름
2. 로봇이 통신으로 정보 받고 쟁반 테이블로 출발
3. 쟁반 테이블 도착 하면 왼쪽 카메라 퍼블리쉬
4. 카메라 실행 후 좌, 우, 중앙 데이터 보내기
5. 데이터 받고 모터 제어로 중앙 맞추기
6. 중앙 맞추면 초음파 퍼블리쉬
7. 왼쪽 90도 회전 후 초음파 전진
8. 전진 마치면 현재 상태 보고 리프트 퍼블리쉬
9. 리프트 마친거 확인 후 쟁반 후진
10. 쟁반 후진 확인 후 각 테이블로 이동
11. 테이블 도착후 해당 방향 카메라 퍼블리쉬
12. 중앙 맞추면 초음파 퍼블리쉬
13. 해당 방향 90도 회전 후 초음파 전진
14. 초음파 마친거 확인 후 현재 상태 보고 리프트 퍼블리쉬
15. 리프트 마친거 확인 후 쟁반 전진
16. 쟁반 전진 확인 후 초음파 퍼블리쉬
17. 초음파 후진 후 해당 방향 90도 회전
18. 로봇 대기 위치로 이동
19. 이동 하면 상태 0번과 1번은 초기화, 상태 2번은 회수하면 초기화
